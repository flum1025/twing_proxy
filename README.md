# TwingProxy

Proxy for Twing
This applicatiton is plugin for [Twing](https://github.com/flum1025/twing).

## Installation

    $ gem install twing_proxy

## Usage

Add configuration

REF: [twing#configuration-file](https://github.com/flum1025/twing#configuration-file)

Example

```yaml
modules:
  proxy:
    endpoints:
      - http://localhost:9292/webhook
```
